Benchmark of Filling A 1,000-item Array in JS and WASM 10,000 times

This benchmark tests the speed of array operations in wasm vs js. Here are some results. Note that this was run on an m1 macbook pro, so expect Safari to run fast. 

CHROME:
    WASM: ~61ms
    JS: ~115ms
SAFARI:
    WASM: ~56ms
    JS: ~65ms
FIREFOX:
    WASM: ~250ms
    JS: ~310ms

Chrome has by far the most stable times, while Safari and FF are less predictable and can change by a 10 or more ms per attempt
